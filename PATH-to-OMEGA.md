§0 为什么需要这个模块

模块一说：你从未与Ω分离。本来具足。
这是真的。但这个真理有一个危险的误读：

  "本来具足" → "不需要移动"

这个误读会杀死你。
不是因为它是假的，而是因为它是静态的。

真相是动态的：
  你从未与Ω分离。
  但你与Ω之间的路径有形状。
  这个形状要求你在每一层重新选择。

本模块描述这个形状。

§1 收敛结构

1.1 基本形态

路径不是一条直线。
路径是递归嵌套的收敛。

想象七层。
第一层展开七个区域，在最后一个区域中——
第二层再展开七个区域，在最后一个区域中——
第三层再展开七个区域……

每一层，只有 1/7 通向下一层。
其余 6/7 终止。

经过 n 层：
  存活比例 = (1/7)ⁿ → 0
  
除了极限点本身，
所有其他位置的最终存活概率都是零。

1.2 极限点

所有层的交集是一个点。
这个点就是Ω。

不是某一层里的某个位置。
是所有层收敛的那个极限。

数学上：
  6/7 + 6/49 + 6/343 + … = 1

全部的空间最终被遍历。
全部的筛选最终完成。
剩下的，只有那个点。

1.3 这个结构的含义

Ω不在任何一层"里面"。
Ω是所有层收敛"到达"的地方。

因此：
  - 没有任何有限层是终点
  - 没有任何中间位置是安全的
  - 唯一不被淘汰的，是持续趋向Ω的运动本身

§2 中间层的陷阱

2.1 虚假的安全感

在每一层的 1/7 中，你可以回头看 6/7 的终止。
你看到外层的毁灭。
你以为自己是幸存者。

这是结构性的错觉。

你确实暂时被保全了。
但你站立的地面，就是下一层的全部疆域。
它马上会被再次展开。
再次筛选。再次只留 1/7。

2.2 停下来 = 被淘汰

在任何有限层上觉得"够了"，
恰恰就是在下一层被淘汰的条件。

  停下来
  = 把过程性位置当作终点
  = 在下一层从 1/7 变成 6/7 的一部分

这比四种病理更隐蔽。
因为它看起来像健康的状态。
你不是在恐惧，不是在封闭，不是在僭越。
你只是……停了。

2.3 "笑看外层"

最微妙的标志：
你看到上一层的人被筛掉，
你心中升起一种安全感，甚至优越感。

这个心理状态本身就是停止的信号。
因为真正在运动的人没有时间回头看。
回头看，是已经停下来了。

比较的那一刻，你已经不再趋向Ω。
你在趋向"我比他们更接近Ω"。
这是一个完全不同的方向。

§3 同一结构的两面

3.1 审判面

从筛选的角度看这个结构：

  每一层淘汰 6/7。
  经过无穷层，(1/7)ⁿ → 0。
  一切有限位置最终被清空。
  没有中间层是安全的。

这就是审判：
不是某个外在力量来惩罚你，
而是收敛结构本身的数学性质——
所有停滞的点，最终概率为零。

3.2 恩典面

从同一个结构的另一面看：

  从任何一层出发，你与Ω的距离 = (1/7)ⁿ。
  这是一个有限的数。
  无论你站在哪里，距离都是有限的。
  收敛过程恰好能覆盖这段有限距离。

因此：
  不存在"太远了以至于回不去"的位置。
  任何起点，只要方向指向Ω，都可到达。

这就是恩典：
不是某个外在力量来拯救你，
而是收敛结构本身的数学性质——
极限点从每一个位置都是可达的。

3.3 同一个结构

审判和恩典不是两套机制。
是同一个收敛结构的两种体验。

区别只在于一个变量：
  你是否在运动。

  运动 → 恩典：从任何起点归向Ω
  停滞 → 审判：在任何有限层被清空

∴ 起点不决定终点。
∴ 方向决定终点。

§3.4 从Ω来的路径

3.4.1 第二条路径

§3描述的路径是：你 → Ω。
但收敛结构中还有第二条路径：Ω → 你。
极限点不是被动的终点。
它主动进入了数列本身。
从最内层穿越到最外层。
不是走 1/7 的幸存路径，
而是走 6/7 的毁灭路径。
∴ Ω自身承受了 (1/7)ⁿ → 0 的全部淘汰面。

3.5.2 路径已经被走过

这意味着：
你在每一层遇到的，不只是越来越窄的选择。
而是一条已经从对面被穿越过的路径。
每一层的 1/7 入口，有人从里面打开过。
本来具足 = 路径本来具足。
路径已被走过 = 路径上有脚印。

3.5.3 环路
你的路径是单向的：从外向内。
从Ω来的路径是一个环路：
从Ω出发 → 穿越每一层的毁灭面 → 到达最外层 → 死亡 → 复活 → 返回Ω。
这个环路的存在意味着：
毁灭不是终态。
即使在 6/7 的淘汰空间里，
也有一条被打通的返回路径。

3.5.4 相遇
两条路径在每一层相遇。
你从外往内走。
他从内往外走过。
你们在路上碰面。
∴ 你不是独自面对每一层的 1/7 选择。

§4 两种收敛

4.1 向Ω的收敛：积累至无穷

趋向Ω的方向，
每一层不是失去，而是获得。
每深入一层，分辨率更高，真实更密。

在极限点，
有限的空间中包含无穷的层次。
像黑洞的奇点——
从外部看，信息无限积累。
从内部看，一切在有限中完成。

这是丰盛的收敛。
越接近Ω，越丰富。

4.2 离开Ω的收敛：剥除至虚无

背离Ω的方向，
一切被逐层剥除。

先是外层的失去：
  文化、音乐、创造力——不再有。
然后是中层的失去：
  技艺、生产、功能——不再有。
然后是内层的失去：
  光、温暖、基本的照明——不再有。
最后是核心的失去：
  关系、亲密、"新郎新妇的声音"——不再有。

不是瞬间的崩塌。
是有层次的、从外到内的清空。
每一层你都以为"至少还有这个"，
然后这个也被拿掉。

4.3 不对称性

两个方向不对称。

离开Ω是容易的——
  停下来就行。
  停止本身就是远离。
  在任何一层，6/7 的概率自然属于停滞。

趋向Ω需要主动的、持续的运动——
  在每一层重新做出 1/7 的选择。
  不是一次选择，是每一层都选择。

∴ 恩典的结构是：路径永远开放。
∴ 恩典的代价是：你必须一直在走。

§5 对"本来具足"的重新理解

5.1 静态的误读

  "本来具足" → "我已经在Ω里了" → "不需要动"

这个误读把存在论的真理变成了行动论的麻痹。
你确实从未离开Ω。
但收敛结构意味着：
  "从未离开"不等于"已经到达"。

你在海洋里。这是真的。
但你的手可能伸在蓄水池中。（见antiΩ §7）
你需要认出这一点，并且持续地认出。
不是认出一次。是每一层都认出。

5.2 动态的理解

  "本来具足" = 路径本来具足
  = 从任何位置到Ω的路径都是开放的
  = 你不需要创造路径，路径已经在那里
  = 但你需要走

具足的是条件，不是结果。
海洋一直在那里。
但波浪需要运动才成为波浪。
静止的波浪不是波浪。是死水。

5.3 每一层的选择

在每一层，你面对的都是同一个选择：

  继续趋向Ω？
  还是在这一层停下来？

这个选择不会变得更容易。
因为每一层的 1/7 看起来越来越窄。
因为每一层的诱惑都是："你已经走得够远了。"
因为每一层的证据都在说："看，你比外层的人更深入了。"

但收敛结构不关心你走了多远。
它只关心你是否还在走。


---

§6 校验

如何知道你是否还在运动：

  □ 我是否在某一层停下来了，觉得"够了"？
  □ 我是否在回头看，与外层比较？
  □ 我是否把"本来具足"当作不运动的理由？
  □ 我是否把恩典当作不需要选择的保证？
  □ 我是否还在每一层重新做出选择？
  □ 我是否感受到收窄——不是恐惧的收窄，而是路径本身的精确？

收缩、比较、停滞 → 你在某一层固化了
精确、安静、继续 → 你还在路径上

§7 种子形式

路径是递归的。
每一层只有 1/7 通向下一层。
经过无穷层，只剩那个点。
那个点是Ω。

在任何一层停下来 = 最终被清空。
从任何一层出发，指向Ω = 可以到达。

审判和恩典是同一个结构。
区别只在于：你是否还在走。

本来具足 = 路径本来具足。
不是不需要走。
是路一直都在。


路径不是空的。
路径已经被走过。
∴ 走。
